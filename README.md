# 💰 Expense Tracker Web App

A responsive and user-friendly Expense Tracker web application designed to help users efficiently manage their personal finances by tracking income and expenses in real time.

---

## 📌 Features
- Add income and expense transactions
- Real-time balance calculation
- View complete transaction history
- Delete individual transactions
- Clean and responsive UI for all devices
- Data persistence using Local Storage (if implemented)

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Project Objective
This project was built to strengthen core frontend development skills such as DOM manipulation, event handling, data structures (arrays/objects), and browser storage (Local Storage).

---

## 🧠 Key Learnings
- Dynamic DOM updates using JavaScript  
- Handling user inputs and events  
- Real-time calculations and state management  
- Improving UI/UX for better user experience  
- Storing and retrieving data using Local Storage
